## 🚀 Perubahan Utama

### jQuery dihapus total (keputusan Wibi, 13 Jul 2026)

Seluruh pemakai jQuery ditulis ulang vanilla JS — bukan sekadar hapus file:

- `public/cube/js/custom.js` — sidebar Cube (treeview slideUp/slideDown,
  restore submenu via localStorage, toggle/pin sidebar, overlay, resize
  handler, card-loader fade) ditulis ulang tanpa jQuery; perilaku 1:1.
- `public/cube/js/overlay-scroll/custom-scrollbar.js` — init scrollbar
  pindah dari plugin jQuery `$().overlayScrollbars()` ke API vanilla
  `OverlayScrollbars(els, opts)`.
- **OverlayScrollbars v1.13.0 vanilla** di-vendor
  (`overlay-scroll/OverlayScrollbars.min.js`, self-hosted — versi sama
  persis dengan CSS existing); `jquery.overlayScrollbars.min.js` dihapus.
- `cubeHead.html`: `<script cube/js/jquery.min.js>` dihapus, swap plugin
  scrollbar ke build vanilla.

### Fix loader (ditemukan saat verifikasi prasyarat 3c)

- `frontend/src/loader.js`: compiler template Vue 3 men-transform asset URL
  relatif (mis. `<img src="../../images/working.gif">` di gov2tablerows)
  menjadi import — loader lama menolaknya sebagai "Bukan SFC" sehingga
  komponen gagal dimuat. Kini import asset dikembalikan sebagai URL string
  (`getFile` + `handleModule` type `.asset`). httpVueLoader lama tidak
  terdampak isu ini (tidak men-transform src), jadi ini regresi 3b yang
  tertangkap & difix di 3c.
- `apps/gov2survey/vue/survey.vue`: hook `destroyed()` (tidak pernah
  dipanggil Vue 3) → `unmounted()` — sisa idiom Vue 2 terakhir.
