## 🚀 Fitur Utama — Dua Jalur Frontend

### Jalur CORE: bundle Vite (baru)
- `frontend/` — Vite 8 + vue@3.5 + mitt + vue3-sfc-loader; build library IIFE
  tunggal → `public/dist/main.js` + `style.css` (**DI-COMMIT** — deploy tetap
  `git pull` tanpa Node, Prinsip #6).
- Komponen inti pindah ke `frontend/src/components/` (Vue 3 Options-compatible):
  gov2table (+tagging/tagged/checkbox/progress/session), gov2pagination,
  gov2notification, gov2search, gov2formfield, gov2button — teregistrasi
  **global** dari bundle; sumber lama di `apps/components/vue/` dihapus.
- `window.phpvb.createPageApp()` — pengganti `new Vue({el:'#app'})` di layout;
  data bridge legacy dipertahankan (vueData/vueComputed/vueMethods/vueCreated/
  vueWatch via Twig, fallback `window.__VUE_DATA__` dkk.).
- `window.eventBus` = mitt + alias `$on/$emit/$off/$once` — kontrak event app
  lama tetap bekerja.
- `src/bv.js` — 17 wrapper `b-*` di atas BS5 native (keputusan #6118:
  TANPA bootstrap-vue-next).

### Jalur APP: runtime-load .vue dipertahankan
- `apps/<app>/vue/*.vue` tetap di-load runtime **tanpa build** via
  **vue3-sfc-loader** (penerus resmi httpVueLoader, author sama) — koreksi
  scope Wibi 12 Jul: pola multi-vue per-app = signature DX phpVB, dipertahankan.
- Konvensi auto-scan PHP (`document.php::component()`) TIDAK berubah.
- Shim `window.httpVueLoader(url)` → adaptor loader baru; fallback path
  per-portal (dulu override `httpVueLoader.httpRequest` di block
  `customVueLoader`) kini perilaku default loader.
- Kompat file lama: `module.exports` → auto-transform ke `export default`.
- **Deviasi sadar:** vue3-sfc-loader di-bundle ke `dist/main.js` (self-hosted),
  bukan file terpisah di `public/js/` — intent "no CDN runtime" tetap terpenuhi.

## 💥 Breaking Changes

- Template head (cube/print) tidak lagi memuat: Vue 2 CDN, `bootstrap-vue.min.js`
  + CSS, `httpVueLoader.js`, `vue-scrollto.js`, Sortable/Vue.Draggable CDN,
  `const eventBus = new Vue()`.
- Mount halaman: `new Vue({` → `phpvb.createPageApp({` (cubeLayout,
  cubePrintLayout, cubeSourceTargetLayout, gov2login layout, errorLayout).
- API Vue 2 yang hilang di Vue 3 tidak tersedia lagi untuk file .vue jalur-app:
  `filters`, `$set/$delete`, `$root.$emit` sebagai bus, `$bvModal`,
  `<template slot=...>` — file .vue bawaan repo sudah dipatch (lihat bawah).

## 🔧 Patch Idiom Vue 2 (file .vue jalur-app)

- `filters` → methods: gov2option-item, gov2controlpanel-item, tablepack (x2).
- `$set()` → assignment reaktif (22 call site): cube-instansi, cube-chat-rag,
  survey, survey-result.
- `<template slot="button-content">` → `#button-content` (tablepack x2).
- `$root.$emit('bv::toggle::collapse')` → `eventBus.$emit` (3 file; wrapper
  `b-collapse` listen via mitt).
- `$bvModal.msgBoxConfirm` → `window.confirm` (gov2option-item,
  gov2controlpanel-item).
- menu-custom: `v-if` pakai variabel `v-for` di elemen sama → dipecah ke
  `<template v-for>` (precedence berubah di Vue 3).
