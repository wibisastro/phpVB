## 💥 Breaking Changes

### Template BS4 `core/template/bootstrap/` dihapus
- Seluruh direktori `core/template/bootstrap/` (bootstrap*.html, bs4*.html, css/) dihapus.
- Path loader Twig kini hanya: `core/template/cube` + `core/template/general`.
- 5 controller yang sebelumnya memakai `b4body.html` jatuh ke default cube:
  - `gov2login/member.php`, `gov2login/privilege.php` → `@gov2login/body.html`
  - `gov2option/controlpanel.php`, `gov2option/option.php`, `gov2option/instansi.php` → `cubeBody.html`
- File view app-local `b4body/b4head/b4header/b4Layout.html` (gov2login, gov2option) dihapus.

### Komponen Vue orphan BS4 dihapus (10 file)
- `gov2login/vue/`: bs4-header, bs4-menu-item, bs4-menu-portal, bs4-menu-settings,
  bs4-notification, bs4roleBrowse, gov2notification-bs4, _bs4tagging, _bs4tagged
- `gov2option/vue/`: gov2notification-bs4
- `gov2login/view/topbar.html` (markup Bulma, orphan)
