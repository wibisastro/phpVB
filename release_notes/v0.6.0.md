## 🆕 Fitur Baru

### gov2wilayah App — Manajemen Wilayah Administratif

Aplikasi baru untuk mengelola data wilayah administrative dengan tiga fitur utama:

#### 1. CRUD Wilayah Hierarkis
- Kelola data wilayah: provinsi → kabupaten → kecamatan → kelurahan
- Tabel recursive dengan tablepack component
- Tombol tambah/edit/hapus per level
- Field validation via JSON config

#### 2. Sidepanel Breadcrumb Navigator
- "Pilih Wilayah" dengan breadcrumb path display
- Search/filter real-time client-side (>10 items)
- List-group styling sesuai Cube theme pattern
- Drill-down dari provinsi hingga kelurahan

#### 3. Topbar Indicator
- Icon + nama wilayah terpilih di header
- Dropdown dengan level breakdown (label kecil, nama besar)
- Click → sidebar terbuka untuk navigasi

#### Session Persistence
- Session keys: `wilayah_id`, `wilayah_nama`, `wilayah_level`, `wilayah_parent_id`
- Perubahan via API endpoint `/gov2wilayah/sidepanel/changeWilayah`

## 🐛 Bug Fixes & Penyempurnaan

### UI Improvements
- **Topbar Positioning**: Dropdown wilayah sejajar dengan profile dropdown (Bootstrap 5 Popper.js)
- **Instansi Revert**: Kembali ke behavior original (tooltip + click→sidepanel), tidak dropdown
- **Sidepanel Filter**: Conditional show hanya jika >10 items, client-side filtering
- **Listbox Styling**: Perindah dengan Cube list-group pattern, breadcrumb 2-level (label atas, nama bawah)

### Navigation Fixes
- **goBack() Logic**: Load breadcrumb untuk item yang diklik (bukan parent-nya)
- **Filter Behavior**: Reset ketika klik item di level baru (tidak stuck di filter level lama)

### Technical Fixes
- Force add `dsnSource.dev.xml` (gitignored by `**/dsn*.xml`)
- Full path URLs di vueData untuk axios interceptor compatibility
