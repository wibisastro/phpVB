> **File markdown belum ada**
>
> Halaman ini mengharap file `{path}` tetapi belum ditemukan di folder `md/` controller.
>
> **Cara mengisi:**
>
> - Buat file pada lokasi `{path}` relatif terhadap controller.
> - Konvensi: nama file mengikuti nama class controller (mis. `dashboard.php` → `md/dashboard.md`).
> - Untuk file alternatif: panggil `$doc->body('readMD', 'nama-file')` agar framework membaca `md/nama-file.md`.
